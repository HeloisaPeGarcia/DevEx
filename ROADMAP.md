# Roadmap

## Desktop App

- Replace simulated provider with GitHub REST API calls.
- Add OAuth or GitHub App authentication.
- Store credentials through Windows Credential Manager.
- Add native Windows notifications.
- Add system tray behavior.
- Move from console to Qt or WinUI 3.

## Platform

- Add an Orbit API control plane.
- Persist environment history in a database.
- Stream workflow logs to the app.
- Add retry and failure recovery.
- Add policy checks before provisioning.

## Infrastructure

- Connect Terraform to a real backend.
- Add Kubernetes namespace quotas.
- Add external-dns and cert-manager integration.
- Add automated TTL cleanup workflow.
- Add image scanning and SBOM generation.

## Product

- Add team dashboards.
- Add cost reports by owner and repository.
- Add GitLab support.
- Add template marketplace.
- Add audit log export.
